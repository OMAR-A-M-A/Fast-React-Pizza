## Description
Fast React Pizza Co. is a modern web application for ordering pizzas. This project demonstrates advanced state management and routing techniques in the React ecosystem. Users can browse a menu, add items to a cart, place an order with GPS location tracking, and prioritize their orders—all without the need for user authentication.

## Features
- **Dynamic Menu:** Loads the pizza menu from a custom API using React Router's loaders.
- **Cart Management:** Add, remove, update quantities, and clear pizzas from the cart.
- **Order Creation & Tracking:** Place an order with user details and track its status using a unique order ID.
- **Geolocation Integration:** Automatically fetches the user's GPS location and physical address for seamless delivery.
- **Priority Orders:** Option to mark an order as "priority" for an additional fee, updating the UI and total price instantly.
- **Optimistic UI:** Modern form submissions and UI updates using React Router actions.

## Tech Stack
- **Frontend Framework:** React (v19)
- **Routing:** React Router (v6.4+ Data Loading API & Actions)
- **State Management:** Redux Toolkit (Global state for the cart)
- **Styling:** Tailwind CSS

> **Note:** This project was built to showcase mastery of modern React patterns, including advanced routing features and global state management.
